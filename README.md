##Books App

## Installation
Welcome to the books app. It has a couple of handy features, you can mark
the books that you have read and store them on several different bookshelves.

If you want to add new books to the shelf, head on over to the search area and search for books.
_of note: Sadly the API is limited and the search terms you can use are also limited. Don't let that stop you from great reading!_

To install it, you must have npm.

`npm install` will install the packages required to view this application.

`npm start` will run the app.
